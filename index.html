Academic Projects

Sentiment Analysis Grammar

Course: NATRUAL LANGUAGE ANALYSIS

Individual Project

Developed a general feature based CFG (Context-Free Grammar) for English using NLTK, the feature grammar was based on Earley's Chart parser. Then built a basic sentiment analysis module based on this grammar and other open source components and enhanced it with ideas from the course.

Technologies: Python, AFINN

Substitution Encryption and Hacking

Course: FOUNDATIONS OF CRYPTOGRAPHY

Group Project group of 4, role: programmer(1/1)

Developed two programs with user-friendly interfaces, one for substitution encryption and one for breaking it using two-gram frequency of letters. The encryption program encrypts the plain text with any non-duplicate key, and the other program automatically breaks the ciphertext produced by the encryption program.

Technologies: Mac OS, Eclipse, Java, Java Swing

Resolution Program to Logic Dots

Course: INTELLIGENT SYSTEMS

Group Project group of 2, role: programmer(1/2)

Developed a resolution program to the game logic dots which is a small game trying to find the right positions of the dots based on the number provided in the column and row. Search the name on the internet and you can see what it is. What we did in the project was writing a program to find the answer of the game. We used two ways to solve the problem respectively which are depth first algorithm and best first algorithm.

Technologies: prolog

Spam Filter

Course: INTRO TO A.I.

Individual Project

Implemented a Naïve Bayes text classifier to use as a spam filter, which involves a training section to add words to the vocabulary with their frequency and calculate their probabilities of both spam and ham. Then comes the classify part, in which calculates both the probabilities of being a ham and a spam for a given email, then adopt the greater one to specify which group this email belongs to.

Technologies: Windows /Mac OS, Eclipse , Java, Naive Bayes
